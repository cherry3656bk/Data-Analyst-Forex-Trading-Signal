# Data Analyst Forex Trading Signal

## Project Overview
This project analyzes forex market data to generate trading insights using historical price data, Monte Carlo simulation scenarios, and trade logs. It focuses on identifying trends, evaluating trading performance, and visualizing market behavior.

## Datasets
- Forex_price_data.csv
- Mc_scenarios.csv
- Trade_log.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Forex price trend analysis
- Trade performance analysis
- Monte Carlo scenario analysis
- Basic risk metrics
- Data visualization
